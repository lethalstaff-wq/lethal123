# Getting Started — с нуля до работающего бота за 10 минут

Этот гайд проведёт тебя от клонирования репозитория до работающего
бота с подключённым ФП-аккаунтом и активной автоматизацией.

## Пререквизиты

- Linux / macOS / Windows с Python 3.11+
- Аккаунт FunPay с хотя бы одним активным лотом
- Telegram-бот от @BotFather (получи токен)
- Опционально: VPS с публичным IP для webhook'ов платежей

## Шаг 1. Клонируем и ставим зависимости

```bash
git clone https://github.com/lethalstaff-wq/lethal123.git
cd lethal123/lethal-fpbot
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Проверяем что установилось:
```bash
python -c "import aiogram; import aiohttp; import aiosqlite; print('ok')"
```

## Шаг 2. Создаём .env

```bash
cp .env.example .env
nano .env
```

Минимум нужно заполнить:
```env
BOT_TOKEN=123456:AA-your-token-from-botfather
ADMIN_IDS=123456789   # твой Telegram ID (узнать через @userinfobot)
```

## Шаг 3. Первый запуск

```bash
python main.py
```

В логах увидишь:
```
INFO База данных инициализирована: lethal_fpbot.db
INFO Applied 5 migration(s)
INFO Loaded 4 plugin(s)
INFO Lethal FunPay Bot запущен. Жду апдейты…
INFO Web сервер поднят на http://127.0.0.1:8080
INFO Запущено фоновых сервисов: 13
```

Жмёшь Ctrl+C чтобы остановить.

## Шаг 4. Подключаем FunPay в боте

Открываем Telegram, пишем боту `/start`. Видим главное меню:

```
📁 Мои аккаунты    | 🔑 Мои прокси
📨 Автоответчик    | 🤖 Автовыдача
📊 Статистика      |
...
```

1. Жмём **📁 Мои аккаунты** → **➕ Добавить аккаунт**
2. Вводим логин FunPay (email или имя пользователя)
3. Вводим пароль (бот удалит сообщение сразу)
4. Прокси (или `-` если без прокси)
5. Ждём «✅ Аккаунт подключён!»

## Шаг 5. Настраиваем автоматизации

### Включаем автоподнятие

`⚙️ Настройки` → жмём 🟢 на `📈 Автоподнятие`. По умолчанию интервал
240 минут (4 часа), поменяй через `⏱ Интервал поднятия`.

### Настраиваем автоответчик

`📨 Автоответчик` → `➕ Добавить триггер`:
- Триггер-слова через запятую: `привет, здравствуйте, hi`
- Ответ: `Здравствуйте! Готов ответить на вопросы по лоту.`

Включаем `📨 Автоответчик` в настройках.

### Добавляем автовыдачу

`🤖 Автовыдача` → `➕ Добавить лот`:
- Название лота (или часть имени): `Steam Random Key`
- Товары (по одному на строку): ключи
- Шаблон сообщения или `-`

Включаем `🤖 Автовыдача` в настройках.

## Шаг 6. Включаем чат через ТГ (Standard+)

Нужна Standard подписка. В настройках включи `🟢 Вечный онлайн` —
это гарантирует что FunPay видит тебя в сети.

Когда покупатель напишет, бот пришлёт сообщение в ТГ с кнопками:
**✍️ Ответить** / **📋 Заготовки** / **🤖 ИИ-ответ** / **🛡 Арбитраж**.

## Шаг 7. Mini App (опционально)

Mini App — веб-интерфейс внутри Telegram. Сборка:

```bash
cd miniapp
npm install
npm run build
```

Сбилженная статика окажется в `miniapp/dist/`, бот сам её раздаёт
по `http://127.0.0.1:8080/app/`. Чтобы открывалось в TG — добавь
Menu Button в @BotFather:

1. /mybots → выбрать бота → Bot Settings → Menu Button
2. URL: `https://your-domain.com/app/`
3. Нужен HTTPS — поставь nginx reverse proxy

## Шаг 8. Деплой

Подробнее в [DEPLOY.md](DEPLOY.md). Коротко:

```bash
# Docker
docker compose up -d

# или systemd
sudo systemctl enable --now lethal-fpbot
```

## Частые проблемы

**«Не удалось войти в FunPay»** — прокси заблокирован Cloudflare,
попробуй другой или без прокси.

**«Сессия протухла»** — нормально, `session_restore` сервис
перелогинится через сохранённый пароль через 10 минут.

**«/healthz возвращает 503»** — проверь что SQLite файл существует и
WAL работает: `sqlite3 lethal_fpbot.db "PRAGMA journal_mode"`.

**Mini App не открывается** — нужен HTTPS. Для локальной разработки:
```bash
cd miniapp
npm run dev
```
и открыть http://localhost:5173
