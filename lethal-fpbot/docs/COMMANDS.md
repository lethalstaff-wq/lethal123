# Справочник команд

Полный список всех команд бота Lethal FunPay Bot. Команды работают
в любой момент — не нужно заходить в меню.

## Основные

| Команда | Описание |
|---|---|
| `/start` | Приветствие и главное меню |
| `/help` | Центр помощи с FAQ |
| `/faq` | Алиас для /help |
| `/menu` | Показать главное меню |
| `/me` | Краткий профиль |

## Настройка

| Команда | Описание |
|---|---|
| `/onboarding` | Чек-лист настройки для новичка |
| `/setup` | Алиас для /onboarding |
| `/suggest` | Персональные советы по улучшению продаж |
| `/tips` | Алиас для /suggest |

## Быстрые действия

| Команда | Описание |
|---|---|
| `/raise` | Немедленно поднять все лоты |
| `/online` | Переключить вечный онлайн |
| `/ping` | Проверить живые сессии |
| `/stop` | Выключить все автоматизации |
| `/go` | Включить базовый набор автоматизаций |
| `/health` | Health check аккаунтов и лотов |

## Статистика

| Команда | Описание |
|---|---|
| `/summary` | Сводка за сегодня |
| `/dashboard` | Графики продаж |
| `/forecast` | Прогноз выручки |
| `/export` | Экспорт данных (CSV, JSON, налоговый) |
| `/crm` | CRM клиентов |

## Pro-инструменты

| Команда | Описание |
|---|---|
| `/lot_gen` | AI-генератор описаний лотов |
| `/templates` | Шаблоны лотов |
| `/broadcast_customers` | Рассылка клиентам CRM по сегменту |

## Админские

| Команда | Описание |
|---|---|
| `/admin` | Админ-панель (только для ADMIN_IDS) |

## Inline-кнопки vs команды

Большинство функций доступны и через красивые inline-кнопки и через
`/команды`. Кнопки лучше для обучения и разовых действий, команды —
для профессионалов которым надо быстро.

## Callback ключи (для разработчиков)

Боту можно слать кастомные callback_data для интеграций:

| Префикс | Назначение |
|---|---|
| `acc:*` | Действия с ФП-аккаунтами |
| `crm:*` | CRM операции |
| `dash:*` | Дэшборд графики |
| `bill:*` | Биллинг |
| `chat:*` | Реакции в чате (reply, ai, arb) |
| `set:*` | Настройки |
| `onb:*` | Onboarding |
| `help:*` | Help Center |
| `sugg:*` | Suggestions |
| `exp:*` | Экспорт |
| `fc:*` | Forecast |
| `tpl:*` | Templates |
| `ad:*` | Auto-delivery |
| `ar:*` | Auto-response |
| `txt:*` | Тексты |
